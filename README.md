# Language Learning Worksheets

**Created by [Mensura Media](https://github.com/MensuraMedia)** &nbsp; | &nbsp; Free to use and distribute

A collection of free language learning worksheets, reference charts, flash cards, and wall prints. All materials include large character references, pronunciation guides, and practice sheets for handwriting practice.

---

## Available Languages

### [Japanese &mdash; 日本語](japanese/)

Worksheets and study tools for learning Japanese writing systems: Hiragana, Katakana, Kanji, and vocabulary topics.

### [English &mdash; 英語](english/)

英語を学ぶための教材。日本語話者向け。
